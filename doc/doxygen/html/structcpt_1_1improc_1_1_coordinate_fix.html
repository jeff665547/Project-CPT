Fix the coordinate system of image to fit the probe spec.

The coordinate system of image and probe spec may not matched, and this step read the user specified image coordinate to rotate or flip the image to match the probe spec, which means the position (0,0) of the probe grid should be the probe which's id is 0.

Parameters
meanThe mean value of intensities grid.
stddevThe standard deviation of the intensities grid.
pixelsThe pixel of raw image
cv_matThe matrix of cv relate to every probe. ( can be empty matrix )
detail_raw_valuesThe detail pixel values in every probe. ( can be empty vector )
roiThe ROI result of ROI process.
origin_positionThe origion position of the image, can be "LT" (left top) or "LB" (left bottom). This is used to fix the coordinate system.
x_axis_directionThe origion position of the image, can be '_' (horizontal) or '|' (vertical). This is used to fix the coordinate system.
